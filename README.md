# my-JSON-server
Using my JSON server to build temporary fake db data for quick back-end development of front-end mock-ups

# Usage
Can only host five endpoints/objects usisng my-josn-server with a limit of 30 items in each endpoint.
Recomend to only host fake data for larger projects and use JSON Placeholder for local development when using real data.
Use this repo to backup fake db data in JSON format.
To extend the library use case host JSON file on private server.
Use Faker.js and Lodash to rapidly generate fake data.
